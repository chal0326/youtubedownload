YouTube Downloader
==================

How to Use:
-----------
1. Double-click the app or run it from Terminal/Command Prompt.
2. When asked, paste the YouTube URL you want to download.
3. Type 'v' for video or 'a' for audio when prompted.
4. The file (video.mp4 or audio.mp3) will be saved in the same folder as the app.

No technical knowledge required!
